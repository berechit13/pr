# 글쓰는 라마 - 모바일 랜딩 페이지

개인 브랜딩을 위한 모바일 친화적 랜딩 페이지 프로젝트입니다. "스스로에게 주는 작은 선물, 한 권의 책을 펴내는데 완벽한 솔루션을 제공합니다"라는 캐치프레이즈를 바탕으로, 글쓰기와 관련된 다양한 주제와 활동을 소개합니다.

## 📌 주요 특징 (Features)

*   **모바일 퍼스트 디자인 (Mobile-First):** 스마트폰 및 태블릿 환경에 최적화된 세로형 스크롤 레이아웃.
*   **모던 UI & 글래스모피즘 (Glassmorphism):** 반투명 카드 UI와 부드러운 그라데이션 배경을 적용하여 현대적이고 세련된 느낌 제공.
*   **동적 스크롤 애니메이션:** `IntersectionObserver` API를 활용하여 사용자가 스크롤을 내릴 때 콘텐츠 카드가 부드럽게 나타나는(Fade-in / Slide-up) 효과 적용.
*   **인터랙티브 CTA 버튼:** 하단에 고정된 '커피챗 신청하기' 버튼을 통해 즉각적인 사용자 행동 유도 (구글 폼 연동).

## 🗂 프로젝트 구조 (Project Structure)

```text
PR/
│
├── index.html   # 메인 HTML 문서 (페이지 마크업 구조)
├── style.css    # 스타일시트 (디자인, 레이아웃, CSS 애니메이션)
├── script.js    # 자바스크립트 (스크롤 인터랙션 및 이벤트 처리)
└── README.md    # 프로젝트 설명 문서
```

## 🛠 사용 기술 (Tech Stack)

*   **HTML5**
*   **CSS3** (Variables, Flexbox, Animations)
*   **Vanilla JavaScript** (DOM Manipulation, IntersectionObserver)

## 🚀 실행 방법 (How to Run)

이 프로젝트는 별도의 서버 구축이나 빌드 과정 없이 웹 브라우저에서 바로 실행할 수 있습니다.

1.  저장소를 클론하거나 프로젝트 폴더를 다운로드합니다.
2.  폴더 내의 `index.html` 파일을 더블 클릭하여 기본 웹 브라우저에서 엽니다.
3.  (선택 사항) VS Code를 사용하는 경우 `Live Server` 확장을 통해 열면 수정 사항을 실시간으로 확인할 수 있습니다.
